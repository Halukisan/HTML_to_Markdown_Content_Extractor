# 提取正文工具接口信息

<img width="1476" height="742" alt="image" src="https://github.com/user-attachments/assets/947fe104-15bd-4bf5-b2b8-48042e03d4c0" />


服务部署到41服务器上，端口号为8765，在线演示地址：http://192.168.182.41:7860/
传入的内容是网页的url和网页的html
调用示例：
```json
response = requests.post(
    "http://192.168.182.41:8765/process",
    json={
        "html_content": html_content,
        "url":"https://www.gov.cn/zhengce/zhengceku/202509/content_7043000.htm"
        }
)
```
输出包含8个字段，
```txt
    status: str
    # 带占位符的结果
    placeholder_html: str = ""
    placeholder_markdown: str = ""
    placeholder_text: str = ""
    placeholder_mapping: str = ""
    # 不带占位符的结果
    cl_content_html: str = ""
    cl_content_md: str = ""
    cl_content_text: str = ""
    # 从extract API获取的其他字段
    html_content: str = ""
    markdown_content: str = ""
    xpath: str = ""
    process_time: float = 0.0
    header_content_text: str = ""
    extract_success: bool = False
```
